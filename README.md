## Technologies

This project was developed using React Native and other libraries such as:

- Expo
- GraphQL
- Apollo GraphQL Client

## Start your Application

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
